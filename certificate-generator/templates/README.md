# 🎓 Certificate Generator System

## 📌 Project Description

This is a full-stack web application that generates professional certificates in PDF format.

## 🚀 Features

* User input form (Name, Role, Event, Date)
* Generate downloadable certificate (PDF)
* Unique Certificate ID
* Admin panel to view records
* Search functionality
* SQLite database integration

## 🛠 Tech Stack

* Frontend: HTML, CSS
* Backend: Python (Flask)
* Database: SQLite
* PDF Generation: ReportLab

## ▶️ How to Run

```bash
pip install flask reportlab
python app.py
```

## 🌐 Routes

* `/` → Generate certificate
* `/admin` → View records

## 📸 Output

* Professional certificate with logo & seal
* Admin dashboard with search
